# Jenkins on EC2

This repository documents the complete step-by-step process of setting up Jenkins on an AWS EC2 instance.

The project covers:
- EC2 instance launch
- SSH connection
- Java installation
- Jenkins installation
- Basic troubleshooting
- Screenshots for reference

This repository is created for hands-on learning and practice in Cloud and DevOps fundamentals using AWS.

Tech stack used:
- AWS EC2
- Linux (Amazon Linux / Ubuntu)
- Java
- Jenkins
- SSH

Folder structure:
- setup/ → installation and configuration steps
- screenshots/ → proof and reference images
- troubleshooting/ → common issues and fixes
